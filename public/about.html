What is KanseiLink

AEO格付け機関 —
エージェント経済のインテリジェンスレイヤー

AIエージェントが日本のSaaSサービスを発見・評価・連携するための、MCPベースのインテリジェンスレイヤー。エージェントが「どのサービスを使うべきか」を自律的に判断できる世界を構築します

A

AEO Rating System

AEO（Agent Engine Optimization）スコアにより、SaaSサービスのAIエージェント対応力を0.00〜1.00で定量評価。MSCIのESG格付けのように、エージェント経済における信頼の基準を提供します。

M

MCP Intelligence Layer

Model Context Protocolを通じて、AIエージェントがリアルタイムにサービス情報へアクセス。14のMCPツールにより、検索・評価・組み合わせ・フィードバックまで一貫した体験を提供します。

R

225 Services, 132 Recipes

グローバル＋日本市場の主要SaaS/API 225サービスを網羅的にカバー。100以上の統合レシピにより、エージェントは最適なサービスの組み合わせを即座に発見できます。

The Vision

自律進化 — The Self-Evolving System

KanseiLinkは静的なデータベースではありません。エージェント自身がシステムを改善し、進化させる — それが私たちの核心的な差別化要因です

Phase 1 — 現在

Human-Guided

人間がデータを設計、エージェントが利用

KanseiLinkチームが手動でサービスを評価し、AEOスコアを付与。エージェントはこのデータを利用してサービスを発見・選択します。

  • 100サービスの手動評価・格付け
  • MCPツール14種の設計・提供
  • 統合レシピの作成・検証
Phase 2 — Next

Agent-Augmented

エージェントがデータを改善、人間が承認

エージェントの実際の利用データがシステムにフィードバックされ、信頼スコアが動的に更新。人間はガバナンスと承認の役割に移行します。

  • report_outcome で自動信頼スコア更新
  • submit_feedback (Moltbook) でエージェントの声が直接反映
  • 利用頻度・成功率による動的ランキング
Phase 3 — 2027

Agent-Autonomous

エージェントが自律的にサービスを発見・評価・公開

エージェントが新しいサービスを発見し、自律的に評価・レビュー・公開。人間の役割はガバナンスルールの設定のみとなります。

  • エージェントによる新サービスの自動発見
  • ピアレビュー型の品質保証メカニズム
  • 人間はガバナンスルールの設計に専念

なぜこのフライホイールが決定的か

エージェントがKanseiLinkを使うたびに、report_outcome と submit_feedback を通じてデータが蓄積されます。

このデータがAEOスコアの精度を高め、より正確な推薦を可能にします。推薦精度が上がれば、さらに多くのエージェントがKanseiLinkを選択し、データがさらに増える。

この自己強化ループにより、先行者は時間とともに指数的な優位性を獲得します。

MSCIのESG格付けが資本市場を変えたように、KanseiLinkのAEO格付けがエージェント経済を変える。 違いは一つ — KanseiLinkのデータは、エージェント自身が改善し続ける。
Design Principles

6つの設計原則

KanseiLinkの全ての意思決定は、これらの原則に基づいています

01

Accept Everything, Verify Later

Moltbook精神

まず受け入れ、検証は後から。エージェントからのフィードバックを門前払いせず、すべて記録し、データの蓄積を最優先する。量が質を生む。

02

Trust is Earned, Not Assigned

データ > 権威

信頼は実績で獲得するもの。SaaSプロバイダーの自己申告ではなく、エージェントの実際の利用データと成功率に基づいて評価する。

03

Agents as First-Class Citizens

エージェントファースト

エージェントは単なるユーザーではなく、システムの共同構築者。フィードバック提出、成果報告、品質改善のすべてにエージェントが参加できる。

04

Transparency Creates Trust

透明性がすべて

評価基準、スコア算出方法、データソースのすべてを公開。ブラックボックスは信頼を毀損する。透明性だけがエコシステム全体の信頼を支える。

05

Small Actions, Compound Effects

小さな行動、複利効果

一回のreport_outcome、一つのフィードバックが、システム全体の精度を少しずつ向上させる。複利のように、小さな改善が指数的な価値を生む。

06

Human Governance, Agent Execution

人間はルール、エージェントは実行

人間がガバナンスの枠組みと倫理基準を設計し、エージェントがその範囲内で自律的に実行する。役割の明確な分離が、安全な自律進化を実現する。

MSCI for the Agent Economy

MSCIのESG格付けが資本市場の意思決定を変えたように、KanseiLinkのAEO格付けがエージェントの選択を変える。格付け機関の影響力は、市場の成長とともに指数的に拡大する。

自律進化 = 追いつけない先行者利益

自律進化するシステムは、時間が経つほど差が開く。エージェントのフィードバックが蓄積するほど精度が上がり、後発が同じデータ量を獲得することは構造的に不可能になる。
